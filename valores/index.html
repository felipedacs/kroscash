Atualizações

As atualizações para "não-adiminisradores" são feitas através dos seguintes passos:

  • Criar um arquivo ".json" por formulário;
  • Enviar o arquivo pelo facebook à equipe.

O banco de dados das atualizações é composto de arquivos ".json" (texto com sintaxe expecífica), podendo ser emitidos por qualquer um aravés deste formulário, porém o mesmo só será cadastrado caso seja upado por nós, administradores do site. Sendo assim, devemos estar cientes do seu arquivo para revisá-lo. Nosso canal de comunicação atual é o facebook, então sempre que tiver um "json novo" de sugestão pode chamar no chat.
